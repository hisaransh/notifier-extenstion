DOM-only monitoring

PR & Jenkins Bell Notifier

This version watches open GitHub PR tabs and open Jenkins pages directly from the DOM. No GitHub or Jenkins API calls, and no tokens.

Settings

Keep the relevant GitHub PR tab or Jenkins page open. The extension only looks at what those pages render in the browser, so it does not poll remote APIs in the background.
